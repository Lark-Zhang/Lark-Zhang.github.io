线性回归

回归(regression)是能为一个或多个自变量与因变量之间关系建模的一类方法。在自然科学和社会科学领域,回归经常用来表示输入和输出之间的关系。

这里以线性回归为例,厘清机器学习的概念.

线性回归的基本元素

price=warea⋅area+wage⋅age+bprice = w_{area} · area + w_{age} · age + b price=warea​⋅area+wage​⋅age+b

www 称为权重weight,决定了每个特征对预测值的影响.
bbb 称为偏置bias、偏移量 offset等.
该线性表达式称为仿射变换 affine transfomation，特点是通过加权和对特征的线性变换 linear transformation 并使用偏置进行平移 translation
对于一个高维数据集，预测公式用矩阵表示法写为：

y^=w⊤x+b\hat{y} = \mathbf{w}^{\top} \mathbf{x}+b y^​=w⊤x+b

损失函数 loss function

对于单个样本,平方误差可以定义为:

l(i)(w,b)=12(y^(i)−y(i))2l^{(i)}(\mathbf{w}, b)=\frac{1}{2}(\hat{y}^{(i)}-y^{(i)})^2 l(i)(w,b)=21​(y^​(i)−y(i))2

为了衡量模型整体质量,需计算损失均值:

L(w,b)=1n∑i=1nl(i)(w,b)=1n∑i=1n12(w⊤x(i)+b−y(i))2L(\mathbf{w}, b)=\frac{1}{n}\sum_{i = 1}^{n}l^{(i)}(\mathbf{w}, b)=\frac{1}{n}\sum_{i = 1}^{n}\frac{1}{2}(\mathbf{w}^{\top}\mathbf{x}^{(i)} + b - y^{(i)})^2 L(w,b)=n1​i=1∑n​l(i)(w,b)=n1​i=1∑n​21​(w⊤x(i)+b−y(i))2

训练模型时,希望能有一组参数(w∗,b∗)(\mathbf{w}^*,b^*)(w∗,b∗),这组参数能最小化总损失,即

w∗,b∗=argminw,b L(w,b)\mathbf{w}^*, b^* = \underset{\mathbf{w}, b}{\text{argmin}} \, L(\mathbf{w}, b) w∗,b∗=w,bargmin​L(w,b)

该优化问题可以用求导方式得到解析解.但深度学习中并不能广泛应用.

随机梯度下降

为减少计算量通常采用随机梯度下降 minibatch stochastic gradient descent

\begin{align*} \mathbf{w} &\leftarrow \mathbf{w} - \frac{\eta}{|\mathcal{B}|}\sum_{i \in \mathcal{B}} \frac{\partial}{\partial \mathbf{w}} l^{(i)}(\mathbf{w}, b) = \mathbf{w} - \frac{\eta}{|\mathcal{B}|}\sum_{i \in \mathcal{B}} \mathbf{x}^{(i)} (\mathbf{w}^{\top}\mathbf{x}^{(i)} + b - y^{(i)})\\ b &\leftarrow b - \frac{\eta}{|\mathcal{B}|}\sum_{i \in \mathcal{B}} \frac{\partial}{\partial b} l^{(i)}(\mathbf{w}, b) = b - \frac{\eta}{|\mathcal{B}|}\sum_{i \in \mathcal{B}} (\mathbf{w}^{\top}\mathbf{x}^{(i)} + b - y^{(i)}) \end{align*}

可以调整但不在训练过程中更新的参数:
∣B∣|\mathcal{B}|∣B∣ 表示小批量中的样本数,或批量大小 batch size.
η\etaη 代表学习率 learning rate
调参 hyperparameter tuning 是选择超参数的过程. 超参数是根据训练迭代结构调整的,训练迭代结果是在独立的验证数据集 validation dataset 上评估得到的.

泛化 generalization

深度学习实践者很少会去花费大力气寻找这样一组参数,使得在训练集上的损失达到最小。
事实上,更难做到的是找到一组参数,这组参数能够在我们从未见过的数据上实现较低的损失,这一挑战被称为泛化(generalization)。
